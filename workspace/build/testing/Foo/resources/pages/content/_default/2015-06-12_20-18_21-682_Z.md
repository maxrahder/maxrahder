As you have seen, you use the +extend+ property in the class descriptor to extend a class.

<pre class="runnable 300">
Ext.define('MyApp.Mammal', {

    extend: 'Ext.Component',
    
    config: {
        name: 'Anonymous',
    }
});
Ext.define('MyApp.taxonomy.Human', {

    extend : 'MyApp.Mammal'

});
</pre>
