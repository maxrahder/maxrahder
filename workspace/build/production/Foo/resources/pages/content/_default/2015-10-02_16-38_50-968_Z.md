A tab panel extends +Ext.Container+, and therefore, has an +items+ array. Each item
is the config for a component.

<pre class="runnable">
Ext.define('MyApp.view.Main', {
    extend: 'Ext.tab.Panel',
    items: [{
        xtype: 'component',
        title: 'Home',
        html: 'Home'
    }, {
        xtype: 'component',
        title: 'Users',
        html: 'Users'
    }]
});
Ext.application({
    name: 'MyApp',
    mainView: 'MyApp.view.Main'
});

</pre>
